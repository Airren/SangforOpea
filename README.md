


# Intel Arc GPU 离线部署大语言模型指南

## 部署环境准备与要求

- Xeon 4th/5th CPU
- BIOS with 'Resizable BAR' enabled
- Ubuntu 22.04.1 LTS with kernel 6.8.0-49-generic(参考文档安装指定Kernel)
- With out-of-tree Intel Arc GPU driver for the best performance
